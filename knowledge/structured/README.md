# Structured Knowledge

Structured knowledge is generated from raw interviews and other raw source material.

This layer converts long-form raw knowledge into organized entities, relationships, timelines, taxonomies, and evidence references. It should make the knowledge base easier to query and generate from without changing the underlying facts.

Structured knowledge must never contain information that does not exist in raw knowledge.

Every structured file should be traceable back to one or more raw interview fragments. If a structured claim cannot be traced to raw source material, it should be removed, corrected, or marked as unsupported until the raw knowledge exists.

The purpose of this layer is structure, not invention.
