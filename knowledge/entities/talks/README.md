# Talks

This folder contains talk entities.

## What Belongs Here

- Presentations, lectures, workshops, panels, interviews, podcasts, and other speaking events.
- Event metadata, audience, topic, contribution, evidence, and related work.
- Links to projects, stories, skills, technologies, publications, companies, roles, and achievements.

## Naming Convention

Use a stable lowercase slug based on the talk title or event:

```text
talk-title.md
```

If needed, include a year or venue:

```text
talk-title-venue-or-YYYY.md
```

## Relationship With Other Entities

Talks show communication and public explanation of knowledge. They should link to the work or skills discussed and keep transcripts or raw source material outside the entity when appropriate.
