# Skills

This folder contains skill entities.

## What Belongs Here

- Professional capabilities that are supported by evidence.
- Definitions, scope, development, and examples of demonstrated use.
- Links to projects, roles, stories, technologies, publications, talks, and achievements.

## Naming Convention

Use a stable lowercase skill slug:

```text
skill-name.md
```

Prefer a specific skill name over a broad category when the evidence supports it.

## Relationship With Other Entities

Skills should be grounded in evidence-bearing entities. They should link to the roles, projects, and stories that demonstrate the skill instead of becoming generic self-description.
