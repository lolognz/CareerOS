# Publications

This folder contains publication entities.

## What Belongs Here

- Articles, papers, reports, essays, books, newsletters, public documents, and similar authored artifacts.
- Metadata, topic, contribution, venue, date, evidence, and related work.
- Links to projects, skills, technologies, talks, companies, roles, and achievements.

## Naming Convention

Use a stable lowercase slug based on the publication title:

```text
publication-title.md
```

If needed, include a year:

```text
publication-title-YYYY.md
```

## Relationship With Other Entities

Publications are externalized knowledge. They should reference the work, skills, and technologies they demonstrate rather than duplicating project or skill content.
