# Knowledge Base

This directory is the canonical source of professional knowledge.

Use it to store structured facts, chronology, stories, relationships, taxonomy, and evidence. Generated outputs should read from this directory, not replace it.

## Main Areas

- `entities/` contains canonical source files for companies, projects, stories, technologies, talks, publications, people, and related concepts.
- `relations/` contains explicit links between entities.
- `indexes/` contains navigation and lookup indexes.
- `taxonomy/` contains shared vocabularies for classification.
- `evidence/` and `media/` can hold supporting source material when appropriate.

Do not invent personal information here. Add facts only when they are known and supported.
